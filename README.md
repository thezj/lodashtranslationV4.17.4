> # util
>> ## attempt(func,[args])
>>> Arguments:  
>>> func(function):需要尝试的函数  
>>> \[args\](...\*):提供给函数使用的参数  
>>> Returns:  
>>> (\*)返回尝试函数的结果或者错误对象  
>>> **使用这个方法执行可能会报错的函数的话，不会由于报错影响后续代码执行，如果函数发生报错会以一个错误对象形式返回，不会阻塞程序运行。**
